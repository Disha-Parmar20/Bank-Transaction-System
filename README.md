# Bank Transaction System

A secure backend banking transaction system built with Node.js, Express.js, MongoDB, and JWT Authentication.

## Features
- User Registration & Login
- JWT Authentication
- Create Bank Account
- Get Account Balance
- Fund Transfer
- Transaction History
- Logout with Token Blacklisting

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Cookie Parser
